# Proyecto Educativo: Virus Médicos

Este proyecto es una página web educativa cuyo objetivo es explicar qué son los virus médicos, cómo se clasifican y cuáles son sus principales características, así como aclarar mitos y realidades relacionados con ellos.

El contenido está orientado a un público estudiantil y utiliza un diseño visual claro, estructurado y fácil de navegar.

---

## Contenido del Proyecto

El sitio web incluye las siguientes secciones:

- Introducción a los virus médicos
- Tipos de virus según el sistema del cuerpo que afectan:
  - Virus respiratorios
  - Virus gastrointestinales
  - Virus transmitidos por la sangre
  - Virus que afectan la piel
- Sección de mitos y realidades sobre los virus
- Información básica sobre prevención

Cada tipo de virus cuenta con su propia página informativa.
